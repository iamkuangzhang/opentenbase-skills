# 会话开始提示

在本项目中开始工作前：

1. 阅读 `PROJECT_BRIEF.md`。
2. 阅读 `dev/CURRENT_STATE.md`。
3. 阅读 `SAFETY.md`。
4. 阅读 `dev/TEST_ENV.md`。
5. 阅读当前技能的 `SKILL.md`。
6. 按需阅读 `skills/system-entry/references/` 下的相关文件。
7. 如果 `dev/evidence/session-notes/` 中已有记录，阅读最新一份。
8. 运行 `git status`。
9. 运行 `git diff`。

不要依赖聊天记忆。项目文件和证据文件才是事实来源。
